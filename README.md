

# Power BI Sales Analysis Project

## Overview
This Power BI dashboard analyzes sales performance using interactive visualizations and KPIs.

## Features
- Total Sales Analysis
- Profit Analysis
- Orders Tracking
- Quantity Analysis
- Top 5 Products
- Bottom 5 Products
- Interactive Slicers

## Tools Used
- Power BI
- DAX
- Data Modeling
- Power Query

## Dashboard Pages
1. Sales Overview
2. Top/Bottom Product Analysis
3. Table Visual Analysis
